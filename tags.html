Important HTML tags

This page is designed as a quick reference for some of the common tags covered in Learn Enough HTML to Be Dangerous. In the process of making it, we'll learn how to make HTML tables via table and related tags.

The tables below don't include all HTML tags, but they do list many of the most important ones.

Block Elements

Tag Name Purpose
h1–h6 headings include a heading (levels 1–6)
table table include a table
tr table row include a row of data
th table header make a table header
td table data include a table data cell

Inline Elements

Tag Name Purpose Example Result
em emphasized make emphasized text <em>technical sophistication</em> technical sophistication
strong strong make strong text <strong>at least a billion people</strong> makes text bold
a anchor make hyperlink <a href="https://www.learnenough.com/">Learn Enough</a> insert a link inside a word or combination of words
img image include an image <img src="https://www.bit.ly/1MZAFuQ" alt="Michael Hartl"> inserts an image into the page
header, div header, div label a part of the page <header><h1>technical sophistication</header></h1><div>at least a billion people</div> abstract semantic tags used to label a part of the page, block elements
span span commonly used to label some words in the page <span>technical sophistication</span> abstract semantic tag used to label some words in the page, inline element
ol ordered list make a list <ol><li>at least a billion people</li></ol> make a list ordered by numbers, need a li tag for each element inside, block element
ul unordered list make a list <ul><li>at least a billion people</li></ul> make a list ordered by dots, need an li tag for each element inside, block element
li list element element inside a list <ul or ol><li>at least a billion people</li></ul or /ol> make an element inside an ordered or unordered list for each li tag, inline element
